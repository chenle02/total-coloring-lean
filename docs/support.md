# Support the project

Total Coloring Lean is open-source mathematical infrastructure. Support helps
with formalization maintenance, documentation, CI and cluster verification,
and reproducible software releases.

[Sponsor Le Chen on GitHub](https://github.com/sponsors/chenle02){ .md-button .md-button--primary }

The repository's native Sponsor button is configured by
`.github/FUNDING.yml`. Sponsorship does not alter theorem statements, claim
boundaries, proof-review standards, or contribution decisions.

## Other valuable forms of support

- Cite the exact software release or commit you use.
- Report documentation drift or a minimal reproducible bug.
- Contribute a small proof, checker, example, or documentation improvement.
- Review a pull request for mathematical scope as well as Lean correctness.
- Star the repository so interested researchers can find it.

Please use GitHub's private vulnerability reporting for security problems and
the issue tracker for ordinary mathematical or documentation questions.
